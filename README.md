# 💊 PharmaTrust — AI-Powered Pharmaceutical Supply Chain

A blockchain-inspired, AI-enhanced pharmaceutical supply chain management system that ensures medicine authenticity from manufacturer to patient.

## 🚀 Features

- **Manufacturer** — Create medicine batches, upload lab reports, generate QR codes for boxes & units
- **Distributor** — Receive stock, forward to retailers, inspect & report issues
- **Retailer / Pharmacy** — Receive from distributor, dispense to patients, raise complaints
- **Patient** — Scan QR code to verify medicine authenticity in real-time
- **Regulator** — Monitor supply chain, view alerts, manage recalls & audit logs
- **AI Sentinel** — Automatically analyzes complaints and flags suspicious activity
- **Offline Verification** — TOTP-based verification without internet
